Q-9 :- How is Cell Padding different from Cell Spacing?

Ans :- Cell padding is the space between the content of a table cell and its border. It is controlled by the cellpadding attribute in the table tag. You can specify the amount of padding in pixels or as a percentage of the cell width. 2. Cell Spacing: Cell spacing is the space between adjacent cells in a table. It is controlled by the cellspacing attribute in the table tag. Like cell padding, you can specify the spacing in pixels or as a percentage of the cell width.


Q-10 :-How can we club two or more rows or columns into a single row or column in an HTMLtable? With Example.

Ans :-In HTML, you can use the colspan and rowspan attributes to combine two or more columns or rows, respectively, into a single column or row in a table. These attributes are applied to the td (table cell) or th (table header) elements within the table. Here's an example of how you can use colspan and rowspan: In this example: 1. he rowspan="2" attribute is applied to the first cell in the first column, indicating that it should span two rows. 2. The colspan="3" attribute is applied to the first cell in the last row, indicating that it should span all three columns.

Ans :-A block-level element always starts on a new line and takes up the full width available. inline element does not start on a new line and it only takes up as much width as necessary.

Q-15 :-How are active links different from normal links?

Ans :- Normal Links: Normal links are links that are in their default state, meaning they haven't been interacted with by the user. They typically have a default appearance specified by the browser or any styles applied in the CSS. Normal links are displayed when the page is initially loaded 2. Active Links: Active links refer to links that are currently being interacted with by the user. This state is often triggered when the user clicks on the link. The appearance of active links can be styled differently to provide feedback to the user that the link is being activated. The styling for active links is often specified using the :active pseudo-class in CSS.

Q-17 :-What is SVG?

Ans :-Unlike other popular image formats like PNG, JPEG, and JPG – which store image information in form of pixels because they are raster-based formats – SVGs store graphics information as a set of points and lines. This means no matter how SVG files are reworked, zoomed, or resized, they don’t become blurred and pixelated like PNG, JPG, and other raster images.


Q-18 :-What is difference between HTML and XHTML?

Ans :-XHTML is basically an extension of HTML, which is stricter than HTML. Both languages are used to create web and Android applications. HTML is SGML based, whereas XHTML is an XML-based language. HTML is easy to learn for beginners, and then they can move on to XHTML.


Q-19 :-What are logical and physical tags in HTML?

Ans :-Physical tag: These tags are used to provide the visual appearance to the text. Logical tag: These tags are used to add some logical or semantic value to the text.
